Intro to Programming: COMP90014 Week 2

In this workshop we'll cover a very little bit of the Unix shell, and then do an introductory programming workshop in Python. This workshop borrows very heavily from Software Carpentry, but we won't have time to cover all the things that Software Carpentry workshops cover. If you want to learn to programming then working through other Software Carpentry lessons is an excellent way to do so.
